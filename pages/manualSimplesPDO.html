Um simples manual caseiro sobre PDO

Eu tinha um pequeno problema, tudo o que eu fazia era em Mysql, com esse cenário eu pensei "Nossa e se do dia para noite precisar trocar o tipo do banco... me ****", deu para entender né, com isso se iniciou uma pesquisa para ver seu eu precisa-va rescrever tudo para outras versões.

Todas minhas aplicações eram PHP + MySQL, então eu descobri o PDO, de forma simples é como um coringa para banco de dados, sua unica necessidade é o driver para acessar o banco.

Claro que ainda necessita reescrever várias coisas, porém vale ressaltar que se no futuro vier a trocar de banco, você não perde o tempo para reescrever, já que já vai estar adaptado para a situação.

Por fim é valido comentar que ele é um pouco mais lento em operaçõe por motivos de um coringa para operações
